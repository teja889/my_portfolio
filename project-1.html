Project Overview

This page contains the case study of Calculator, which includes the Project Overview, Tools Used and Live Links to the official product.

As part of my project, I created a custom calculator using HTML, CSS, and JavaScript. The calculator is designed with a clean and user-friendly interface. It features a standard layout with numeric buttons, arithmetic operators, and a display screen to show the input and result. And it performs basic operation like addition, subtraction, division, multiplication.
To enhance the user experience, I incorporated CSS styles to make the calculator visually appealing. Using JavaScript, I implemented the logic for performing calculations based on user input.Overall, my custom calculator project showcases my proficiency in HTML, CSS, and JavaScript, demonstrating my ability to create functional and visually appealing user interfaces.
